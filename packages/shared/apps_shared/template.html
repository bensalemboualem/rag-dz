Template IAFactory

Ce template utilise les composants unifiés IAFactory Algeria.
Header, Footer et Chatbot sont automatiquement injectés.
Supports: Dark/Light mode, i18n FR/AR/EN, RTL pour l'arabe.

Fonctionnalités incluses:

  • Header avec logo animé et navigation
  • Toggle User/Développeur
  • Sélecteur de langue FR/EN/AR avec RTL
  • Liens sociaux (GitHub, Hugging Face)
  • Boutons d'authentification (Log in, Get Started)
  • Menu mobile responsive
  • Footer avec 10 réseaux sociaux
  • Chatbot Dzir IA avec 3 modes (Chat, RAG, Support)
  • Dark/Light mode persistant (localStorage)

Comment utiliser:

<!-- 1. Inclure les fichiers CSS et JS -->
<link rel="stylesheet" href="/apps/shared/iafactory-unified.css">
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.1/css/all.min.css">
<script src="/apps/shared/iafactory-unified.js"></script>

<!-- 2. Ajouter data-iaf-auto-init au body pour auto-init -->
<body data-iaf-auto-init>

<!-- 3. Ajouter les placeholders pour les composants -->
<div id="iaf-header" data-iaf-header></div>
<!-- Votre contenu ici -->
<div id="iaf-footer" data-iaf-footer></div>
<!-- Le chatbot est auto-injecté -->

API JavaScript:

// Changer la langue
IAFactory.setLanguage('fr'); // 'fr', 'en', 'ar'

// Changer le thème
IAFactory.toggleTheme();

// Ouvrir/fermer le chatbot
IAFactory.toggleChatbot();

// Changer le mode chatbot
IAFactory.setChatMode('chat'); // 'chat', 'rag', 'support'

// Écouter les changements de langue
window.addEventListener('languageChanged', (e) => {
    console.log('New language:', e.detail.lang);
});

// Écouter les changements de profil (User/Dev)
window.addEventListener('profileChanged', (e) => {
    console.log('New profile:', e.detail.profile);
});
